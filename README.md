# Search Bookmarks Fast

## Features

-   **Instant Search**: Quickly find bookmarks by title or URL.
-   **Omnibox Support**: Type `bm` followed by your query in the address bar to search directly.
-   **Smart Context Menu**: Right-click on any page to "Save to Search Bookmarks Fast..." with intelligent folder suggestions based on your usage.
-   **Keyboard Shortcuts**: Use `Ctrl+Shift+S` (Windows/Linux) or `Command+E` (Mac) to open the search popup instantly.
-   **Folder Navigation**: Easily see the folder path for each bookmark.

## Installation

1.  Clone or download this repository to your local machine.
2.  Open Google Chrome and navigate to `chrome://extensions/`.
3.  Enable **Developer mode** in the top right corner.
4.  Click **Load unpacked**.
5.  Select the directory where you downloaded/extracted this extension.

## Usage

### Popup Search
Click the extension icon in the toolbar or use the keyboard shortcut (`Ctrl+Shift+S` / `Cmd+E`) to open the search popup. Start typing to filter your bookmarks instantly.

### Omnibox (Address Bar)
Type `bm` in the address bar and press **Space** or **Tab**. Then type your search query to see matching bookmarks directly in the suggestions list.

### Saving Bookmarks
Right-click on any webpage and select **Save to Search Bookmarks Fast...**. The extension will suggest the most relevant folders for you to save the bookmark into, based on your previous habits.
